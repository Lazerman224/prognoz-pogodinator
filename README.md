# Приложение для прогноза погоды (Вариант 3)

## Как запустить

### В Visual Studio (F5)
1. Создайте новый проект Console Application.
2. Замените код на содержимое weather_forecast.cpp
3. Нажмите F5.

### Через командную строку
g++ weather_forecast.cpp -o weather
./weather
